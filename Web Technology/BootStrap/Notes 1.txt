BootStrap : Is a free Frontend framework for faster and easier web development
  - It includes Html,Css based design templates for typography,forms,buttons,tables,navaigation,modals,images,carousels and many more other options, as well as java script plug-ins
  - It also gives the ability to create responsive designs
  - Was developed by Mark Otto and Jacob in the year 2011
  - Is an Open source framework targetting Mobile-first approach
  - Easy to Use
  - Browser Compatability

- Always include the Html5 doctype

eg :<!Doctype html>

- Inorder to ensure proper rendering, touch and zoom, we need to add a <meta> tag inside the <head> 
<meta name="viewport" content="width=device-width,initial-scale=1">

Containers : all site contents have to be wrapped in a containing element
  - are of 2 types
 1: .container class - a responsive fixed container
    .container-fluid  - full width container,spanning the entire width of the viewport

Components: A set of reusable and customizable components to make web development faster and easier
- List of Components:
1. Jumbotron: making any content look larger and bigger to catch the attention of the user
2. Alerts: Predefined Popup messages
3. Buttons:perform actions on the form,dialog box. Multiple states, size and styles
4. Badge : Labelling component
5. Progress bar : Progress of some operation
6. Nav bar: The header bar at the top of the application and provides navigation thru menus
7. Forms: Multiple Inputs
8. List Group : provides display of unorder series of contents in aproper way
9. Card :Provides customizable,extensible and flexible content container
10. Carousel : slide of images using built in 3d css and javascript
11. Breadcrumbs: Provides location of the current page in the navigational hierarchy and also has seperators thru css
12. Pagination : Navigation between different pages
